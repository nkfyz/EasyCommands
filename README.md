# EasyCommands

Welcome to EasyCommands, a tool for IDE-style autocompletion across multiple oprating systems.

## Setup

Install deps:

```bash
cd EasyCommands/
npm install .
```

## Usage

```bash
# within EasyCommands dir
npm i -D tsx
npx tsx src/index.ts
```

type `git` and a space to see the options, enjoy!
